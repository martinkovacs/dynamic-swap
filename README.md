# Dynamic swap
This is a really simple script for dynamic swap management on linux.

# Dependencies

 - bash
 - bc
 - awk / gawk

**Debian / Ubuntu:**

    sudo apt install bash bc gawk

**Arch**

    sudo pacman -S bash bc gawk

# Install

- `git clone https://github.com/martinkovacs/dynamic-swap`
- `cd dynamic-swap`
- `chmod +x *`
- `./install.sh`
